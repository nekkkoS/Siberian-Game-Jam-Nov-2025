# The Last Fragments (Project for Siberian Game Jam November 2025)
**Truth hurts more than fear.** <br>
A short psychological horror about the burden of remembering. <br>

Controls: <br>
W - Move forward <br>
A - Move Left <br>
S - Move Right <br>
D - Move Backward <br>
Esc, P - Pause <br>
Space - Action <br>
