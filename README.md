# The Last Fragments (Project for Siberian Game Jam November 2025)
**Truth hurts more than fear.**
A short psychological horror about the burden of remembering.

Controls:
W - Move forward
A - Move Left
S - Move Right
D - Move Backward
Esc, P - Pause 
Space - Action
